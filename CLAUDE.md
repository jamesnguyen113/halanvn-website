# halanvn.com — Hà Lan Corporation website

Static Astro + Tailwind v4 site for the family steel business (tôn lợp, xà gồ C/Z, thép).
Vietnamese primary, minimal English pages under `/en/`.

## Deploy — IMPORTANT
- Live at https://halanvn.com via **GitHub Pages** (repo: jamesnguyen113/halanvn-website).
- **Every push to `main` auto-deploys in ~2 min** (.github/workflows/deploy.yml). No manual deploy step.

Never touch the website `A`/`CNAME`/`TXT` records (GitHub Pages + Google verification).
- `public/_redirects` is inert on GitHub Pages; old-URL redirects are static meta-refresh pages generated by `gen-redirects.mjs` into `public/vn/`. If a redirect changes, edit `public/_redirects` then run `node gen-redirects.mjs`.

## Where content lives
- Prices (Bảng giá): `src/data/prices.ts` — rows `{ spec, weight?, price }`, `price: null` renders "Liên hệ". Update `priceListUpdated` when changing prices.
- Phones/addresses/factories/Zalo/email: `src/data/site.ts`
- Products: `src/data/products.json` (+ categories/EN names in `src/data/products.ts`)
- Page copy: `src/pages/*.astro` — plain Vietnamese text inline
- Images: `public/img/`, referenced as `/img/...`
- Old-site archive (do not edit): `content-archive/`

## Conventions
- Currency: VND with vi-VN separators (`fmtVnd`), e.g. 57.000 đ
- Colors/fonts are Tailwind theme tokens in `src/styles/global.css` (brand red #b91c1c, steel grays). Use `text-brand-soft` for red text on dark backgrounds (contrast).
- Business converts by phone: keep call/Zalo CTAs prominent. Mobile-first.
